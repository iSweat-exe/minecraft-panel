fix(panel): correct FileAction enum serialization format

- Fixed invalid enum variant errors (`unknown variant 'Delete'`) when calling `nodeFileAction`.
- The `FileAction` Rust enum is annotated with `#[serde(rename_all = "snake_case")]`, so unit variants must be passed as `"delete"` or `"mkdir"`, and structured variants as `{ rename: { new_name: ... } }`.
- Fixed `SftpPanel.tsx`, `SftpFileList.tsx`, and `ModsPanel.tsx` to pass the correctly formatted actions.
